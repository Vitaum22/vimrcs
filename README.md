# Instalar
sh -c "$(curl -fsSL https://raw.githubusercontent.com/LeonardoFerreiraa/vimrcs/master/install.sh)"

# Atalhos
```
F2 :NERDTreeToggle
F3 :NERDTreeFind
F5 :GundoToggle
CTRL + L :nohls
```
